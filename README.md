# HIBA JASSOUSTI - Architectural Portfolio

![Hiba Jassousti Hero Image](public/images/hero%20section.png)

## Overview

Welcome to the digital portfolio of **Hiba Jassousti**, an architectural designer specializing in the fusion of heritage preservation and contemporary, sustainable design. This premium, bilingual (English/French) web portfolio is built to reflect a luxury editorial aesthetic, highlighting Hiba's strategic expertise, professional background, and architectural philosophy.

Her design approach is deeply rooted in the concept of *Genius Loci* — the spirit of the place. From analyzing the traditional typology of the Djerbian Menzel to modern coastal residences, this portfolio showcases a body of work that prioritizes ecological harmony, cultural memory, and sensory experience.

## Features

- **Bilingual Interface (English & French)**: Full internationalization setup using React Context and a custom localized dictionary.
- **Luxury Editorial Design**: Tailored color palettes, sophisticated typography (Cormorant Garamond & Inter), and micro-interactions.
- **Fluid Animations**: Smooth page transitions, scroll-driven reveals, and dynamic elements powered by Framer Motion and GSAP.
- **Immersive Galleries**: Interactive masonry layouts and full-screen image lightboxes for architectural visualization.
- **Responsive Architecture**: Fully optimized for desktop, tablet, and mobile viewing.

## Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion & GSAP
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
